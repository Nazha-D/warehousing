# Multi-Tenant Warehouse Management System (WMS)

A robust Backend system built with **Laravel** to manage complex warehouse operations for multiple companies independently.

## 🚀 Key Features
- **Multi-tenancy:** Complete data isolation between companies using Laravel Policies and Permissions.
- **Inventory Management:** Full CRUD for products, categories, and groups.
- **Stock Control:** Intelligent quantity calculation based on replenishment and inter-warehouse transfers.
- **Financial Module:** Handling taxation groups, discounts, and automated invoicing.
- **Automated Reporting:** Detailed reports for stock levels and transactions.
- **Admin Dashboard:** A centralized control panel for system-wide administration.

## 🛠 Tech Stack
- **Framework:** Laravel 10/11
- **Language:** PHP 8.x
- **Architecture:** Clean Code, Service Layer (In Progress), Multi-tenancy.
- **Security:** Role-Based Access Control (RBAC).
